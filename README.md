# is-legal

`is-legal` is a package that determines if a person is of legal age.

## Installation

You can install the package using npm:

```bash
npm install is-legal
```

## Usage

```javascript
import { is } from 'is-legal';

const age = 20;
console.log(is(age).legal()); // true
```
